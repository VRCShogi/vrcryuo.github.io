# VRC竜王戦 公式サイト

GitHub Pages でそのまま公開できる、1ページ構成の静的サイトです。

## 特徴

- HTML / CSS / JavaScript のみで構成
- サーバーサイド処理なし
- 相対パスで動作
- `棋譜/index.json` から棋譜一覧を取得
- `.kif` を読み込んで盤面・持ち駒・指し手一覧を表示
- `.kif` ダウンロード対応
- 和風・木目調のレスポンシブデザイン

## 採用方針

### なぜ外部ライブラリを使っていないか

今回の実装では、GitHub Pages での配置を最小構成にし、将来的な保守コストを下げるために外部ライブラリを使っていません。

その代わり、`script.js` に以下を実装しています。

- `棋譜/index.json` の読込
- KIF の簡易パーサー
- 平手初期局面の生成
- 指し手に応じた局面再現
- 将棋盤・持ち駒・指し手一覧の描画
- 手数移動操作

### 盤面表示の対応範囲

- `.kif` 読み込み: 対応
- 平手: 対応
- 指し手の局面再現: 対応
- 成り / 打つ / 同: 対応
- 投了などの終局語: 対応

このサンプル実装では、**手合割は平手のみ対応**です。`script.js` を拡張すれば、香落ちなどの初期配置も追加できます。
