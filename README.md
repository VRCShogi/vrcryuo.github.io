# VRC竜王戦 公式サイト

VRC竜王戦のイベント情報、リーグ勝敗表、棋譜ビューア、関連リンクをまとめた GitHub Pages 向けの静的サイトです。

HTML / CSS / JavaScript だけで構成しているため、ビルド手順やサーバーサイド処理はありません。`index.html` を入口に、`script.js` が `棋譜/index.json` と `.kif` ファイルを読み込んで棋譜ビューアを動かします。

## 主な機能

- イベント概要の掲載
- AからDグループのリーグ勝敗表とタブ切り替え
- `棋譜/index.json` に登録した棋譜一覧の表示
- `.kif` ファイルの読み込み
- 将棋盤、持ち駒、指し手一覧の表示
- 初期局面、1手戻る、1手進む、最終局面への移動
- 表示中の `.kif` ファイルのダウンロード
- Discord、VRChat Group、関連イベントへのリンク掲載
- PC / スマートフォン向けのレスポンシブ表示

## ローカルで確認する

`.kif` や `index.json` は `fetch()` で読み込むため、ブラウザで `index.html` を直接開くより、静的ファイルサーバー経由で確認するのがおすすめです。

Python が使える環境では、リポジトリ直下で次のように起動できます。

```powershell
python -m http.server 8000
```

起動後、ブラウザで次の URL を開きます。

```text
http://localhost:8000/
```

## GitHub Pages で公開する

このリポジトリは静的サイトなので、GitHub Pages の公開元にリポジトリ直下を指定すればそのまま動作します。

- 入口ファイル: `index.html`
- スタイル: `styles.css`
- 画面の動きと棋譜ビューア: `script.js`
- 画像: `images/poster.png`
- 棋譜一覧: `棋譜/index.json`
- 棋譜ファイル: `棋譜/**/*.kif`

## 棋譜を追加する

1. `.kif` ファイルを `棋譜/グループA/` など任意のディレクトリに追加します。
2. `棋譜/index.json` の `sections` に表示名とパスを登録します。
3. ローカルサーバーでページを開き、棋譜一覧と盤面表示を確認します。

登録例:

```json
{
  "name": "GroupA",
  "files": [
    {
      "title": "○先手名 vs 後手名●",
      "path": "棋譜/グループA/example.kif"
    }
  ]
}
```

`path` は `index.html` から見た相対パスです。JSON 内では Windows の `\` ではなく `/` を使ってください。

## 棋譜ビューアの対応範囲

`script.js` には簡易的な KIF パーサーと局面再現処理を実装しています。

- 手合割は `平手` に対応
- `先手` / `後手` / `下手` / `上手` のメタ情報に対応
- `開始日時`、`棋戦`、`手合割` の読み取りに対応
- 通常の指し手、成り、打つ、同、投了などの終局語に対応
- 持ち駒と成駒の表示に対応

香落ちなど平手以外の手合割は、現在の実装ではエラー表示になります。対応する場合は、`script.js` の初期局面生成処理を拡張してください。

## 更新時の主な編集先

- イベント概要、勝敗表、関連リンクを変える: `index.html`
- 色、余白、レスポンシブ表示を変える: `styles.css`
- 棋譜一覧の読み込み、KIF 解析、盤面表示を変える: `script.js`
- サイトに表示する棋譜を増減する: `棋譜/index.json`
- ポスター画像を差し替える: `images/poster.png`

## 注意点

- `棋譜/index.json` に登録されていない `.kif` は、ファイルが存在していてもサイトの一覧には表示されません。
- ファイル名やパスを変更した場合は、`棋譜/index.json` の `path` も合わせて更新してください。
- 日本語のディレクトリ名とファイル名を使用しているため、移動やアップロード時は文字コードとパスの変更に注意してください。
